# **django-movie**

This is a test project, or rather a site that publishes films of different genres.

## **Frameworks:**
  - Django
  - Django-rest-framework
  - Simple-JWT
  - Ckeditor
  - Ckeditor_uploader
  - Snowpenguin
  - Allauth
  - Django_filters
  - Djoser
  - Drf_yasg
  - Oauth2_provider
  - Social_django
  - Rest_framework_social_oauth2
  - Corsheaders

## **Databases:**
  - SQLite3
  - PostgreSQL
